# color-conversion

## Purpose

sRGB 与 CIELab 色彩空间之间的相互转换，以及 RGB 到灰度的转换。

## Requirements

### 需求：将 sRGB 图像转为 CIELab 色彩空间

系统应使用标准 sRGB→线性→XYZ→Lab 转换链，以 D65 为参考白点，将 [0,1] 范围 RGB 值转为 CIELab。
转换应使用 sRGB gamma 传递函数和 CIE 1931 标准观察者。

#### 场景：白色映射到 Lab 中的 (100, 0, 0)
- **当** RGB=(1.0, 1.0, 1.0)
- **则** Lab 应约为 (100.0, 0.0, 0.0)

#### 场景：黑色映射到 Lab 中的 (0, 0, 0)
- **当** RGB=(0.0, 0.0, 0.0)
- **则** Lab 应约为 (0.0, 0.0, 0.0)

#### 场景：纯红色具有正 a* 值
- **当** RGB=(1.0, 0.0, 0.0)
- **则** L 应约 53，a* 应为正，b* 应为正

### 需求：将 CIELab 图像转回 sRGB

系统应使用逆向 Lab→XYZ→线性→sRGB 链将 CIELab 值转回 [0,1] 范围的 RGB。
往返转换在浮点容差内有损。

#### 场景：往返保留中性灰
- **当** RGB=(0.5, 0.5, 0.5)
- **则** RGB→Lab→RGB 后，所有通道应在 0.5 的 1e-6 以内

#### 场景：往返保留饱和色彩
- **当** RGB=(1.0, 0.0, 0.0)
- **则** RGB→Lab→RGB 后，所有通道应在原始值的 1e-4 以内（色域边缘色彩在 sRGB→线性→XYZ→Lab 往返中会累积浮点误差）

#### 场景：越界 Lab 值被钳制
- **当** Lab 值产生负的线性 RGB
- **则** 输出 RGB 应被钳制到 [0.0, 1.0]

### 需求：将 RGB 图像转为灰度

系统应使用 ITU-R BT.601 亮度系数将 RGB 转为灰度。
公式：`gray = 0.299*R + 0.587*G + 0.114*B`

#### 场景：白色 RGB 变为白色灰度
- **当** RGB=(1.0, 1.0, 1.0)
- **则** 灰度值应为 1.0

#### 场景：黑色 RGB 变为黑色灰度
- **当** RGB=(0.0, 0.0, 0.0)
- **则** 灰度值应为 0.0

#### 场景：相等 RGB 通道产生相同灰度值
- **当** 所有像素的 R=G=B
- **则** 灰度值应等于任一输入通道的值
