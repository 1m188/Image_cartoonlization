# Image_cartoonlization_go

根据 [Image_cartoonlization](../) 项目做的golang的移植

Image_cartoonlization 是我当年还在读大学的时候，某一个学期的matlab课程作业，要求使用matlab做一个什么东西出来，当时兜兜转转在知乎上找到了github上的这个项目，然后顺利通过了，由于课程老师刚好是我们大学生创新项目的导师，所以也没有什么波折。

最近一直在使用AI agent工具，opencode + deepseek-v4-pro 做了不少东西，顺手的很，属于是手里有个锤子看什么都像钉子，无意中在github上曾经star过的项目集合中看到了这个，于是想着用golang重写一下，刚好也能够跨平台，于是用ai几十分钟就捣鼓出来了，ai还是太好用了。

看到这个项目又情不自禁想起当年和同学一起搞大学生创新创业项目的时光，今年距离当年我高考也差不多过了十年了，回首望去竟然一下子有些想不起来自己曾经做过什么事情，为什么会在这里，小时候总觉得十年是非常漫长的时间，毕竟过个暑假对于我而言当时都算是煎熬，可是现在看来，真是白驹过隙，落花流水。距离大学毕业也都有六年了，不知道当年的同学队友，如今是否安好？

就像朴树唱的那句歌词一样：

“散落在天涯”
